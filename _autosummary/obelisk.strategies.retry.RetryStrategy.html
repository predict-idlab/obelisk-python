obelisk.strategies.retry.RetryStrategy¶

class obelisk.strategies.retry.RetryStrategy[source]¶

Bases: ABC

Base class for all retry strategies, whether predefined or user-made. The strategy has as its sole purpose to create instances of RetryEvaluator that can be used for a specific operation.

Each individual failable operation will request a separate evaluator using the make method.
