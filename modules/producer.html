Obelisk producer.

class obelisk.producer.ObeliskProducer(client_id, client_secret, debug: bool = False)[source]

Component that contains all the logic to produce data to the Obelisk API. Obelisk API Documentation: https://obelisk.ilabt.imec.be/swagger/

send(dataset: str, data, precision: TimestampPrecision = 'MILLISECONDS', mode: IngestMode = IngestMode.DEFAULT)[source]

Ingest data to Obelisk.

Parameters:

  • dataset – The ID of the dataset to upload the events to.

  • data – Array of data points (see Obelisk docs)

  • precision – Determines how the UTC timestamps for the Metric Events should be interpreted.

  • mode – mode of ingestion in Obelisk - can either be ‘default’ (data for both storing and streaming), ‘stream_only’ or ‘store_only’

Returns:

requests.Response
