events(datasets: list, metrics: Optional[list] = None, precision: TimestampPrecision = 'MILLISECONDS', fields: Optional[dict] = None, from_timestamp: Optional[int] = None, to_timestamp: Optional[int] = None, order_by: Optional[dict] = None, filter_: Optional[dict] = None, limit: Optional[int] = None, limit_by: Optional[dict] = None, cursor: Optional[str] = None)[source]

Query historical events for the specified Metric.

Parameters:

  • datasets – List of Dataset IDs.

  • metrics – List of Metric IDs or wildcards (e.g. *::number).

  • precision – Defines the timestamp precision for the returned results.

  • fields – List of fields to return in the result set. Defaults to [metric, source, value]

  • from_timestamp – Limit output to events after (and including) this UTC millisecond timestamp.

  • to_timestamp – Limit output to events before (and excluding) this UTC millisecond timestamp.

  • order_by – Specifies the ordering of the output, defaults to ascending by timestamp.

  • filter – Limit output to events matching the specified Filter expression.

  • limit – Limit output to a maximum number of events. Also determines the page size. Defaults to 2500.

  • limit_by – Limit the combination of a specific set of Index fields to a specified maximum number.

  • cursor – Specifies the next cursor, used when paging through large result sets.

sse(name: ~typing.Optional[str] = None, datasets: ~typing.Optional[list] = None, metrics: ~typing.Optional[list] = None, stream_id: ~typing.Optional[int] = None, receive_backlog: bool = False, **kwargs) -> (<class 'str'>, <class 'sseclient.SSEClient'>)[source]

Create and/or open an event Stream for the Dataset. If stream_id is not provided, a new stream will be created using the name, datasets and metrics.

Parameters:

  • name – Name for the Stream.

  • datasets – List of Dataset IDs for the Stream dataRange.

  • metrics – List of Metric IDs or wildcards (e.g. *::number) for the Stream dataRange.

  • stream_id – The id of the Stream.

  • receive_backlog – Whether to receive events that were missed since last time the client was connected. If set to ‘false’ (default), Obelisk starts streaming live data immediately.

Returns:

stream_id, SSEClient
